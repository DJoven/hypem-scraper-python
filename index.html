Hypem-scraper-python

Hypem-scraper-python was forked from a JavaScript/Python work-in-progress that would ideally allow a user to download all of their favorited music on Hypem. The JavaScript portion has been removed because it was incomplete and ineffective. This is just the python script reworked so that it will scrape a specified user's Hype Machine favorites and download each respective sound cloud mp3. The songs do not always match up, however the program is about 95% accurate.

Requirements

The python script requires that you have at least Python 3.3.0, requests and simpleJSON installed.

  • Python 3.3.0
  • requests
  • simpleJSON

Usage

Run "scrape_songs.py" using the Python shell.

Type the following, replacing "username" with the desired username, in to the Python shell:

get_all_songs("username")

Each song will be downloaded to the same folder that "scrape_songs.py" is in.
